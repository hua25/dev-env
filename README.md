# dev-env
使用Docker构建基础开发环境,如:DB、MQ等。

- 为什么会有这个项目？
> 因为**懒**。

每次需要搭建新的开发环境时，DB、MQ…… 一堆东西，安装、配置，太浪费时间，还好有了Docker，免去了很多搭建开发环境是软件安装安装的麻烦。整理一下搭建环境是常用的数据库、中间件之类的东西在Docker环境下的使用。
